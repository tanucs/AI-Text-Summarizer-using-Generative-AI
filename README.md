# AI Text Summarizer using Generative AI
![Text Summarizer Banner](https://github.com/tanucs/AI-Text-Summarizer-using-Generative-AI/blob/main/OIP.webp)

## Overview
This project demonstrates the use of prompt-based interaction with a generative AI model to create concise summaries from long-form text.

## Objective
To understand how prompt design and instructions influence the quality and structure of AI-generated summaries.

## Approach
- Designed prompts to summarize long text into concise outputs
- Experimented with different prompt formats and instructions
- Compared results to improve clarity and relevance

## Key Learnings
- Prompt structure significantly impacts AI output quality
- Clear instructions lead to more usable summaries
- Generative AI can be applied effectively without model training

## Tools Used
- Generative AI concepts
- Prompt Engineering
- Text-based input/output
